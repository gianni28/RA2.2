# RA2.2 – Liquid Tank Level Monitoring

**Course:** Automation and Process Control  
**Student:** Giovanni Andrea Raffa Fiorello  
**University:** Universidad de La Sabana  
**Date:** April 2026  

## About this project
This project implements a combinational Ladder logic program to monitor the level of a chemical liquid tank using three binary sensors and five light indicators. The solution was developed and simulated in CODESYS, then validated on real hardware using OpenPLC and an Arduino UNO R3.
